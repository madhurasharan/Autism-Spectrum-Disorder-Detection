# 🧠 Early Autism Spectrum Disorder Detection

Machine learning web application to predict Autism Spectrum Disorder (ASD) using clinical and behavioral data with high accuracy.

---

## 📌 Overview

This project predicts the likelihood of Autism Spectrum Disorder (ASD) using machine learning models trained on real-world clinical and behavioral data.

It aims to assist in early screening and enable faster, data-driven decision-making.

---

## 🧠 Model & Results

- **Stacking Ensemble Model:**
  - XGBoost
  - Random Forest
  - CatBoost

- **Performance:**
  - 95% accuracy (training dataset)
  - 94% accuracy (external dataset)

---
